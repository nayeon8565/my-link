# 사용자 시나리오 (User Scenarios) - 마이링크

## 1. 방문자: 마이링크 페이지 방문 과정
- 방문자 사용자는 **크리에이터의 다양한 작업물과 활동 채널을 한곳에서 확인하기 위해** 인스타그램 등 SNS에 공유된 고유 접속 URL(`mylink.com/닉네임`)을 클릭한다.
- 방문자 사용자는 **이 페이지의 주인이 누구인지 명확히 파악하기 위해** 화면 최상단에 노출된 구글 기반 프로필 이미지, 유저네임(실제 활동명), 그리고 한 줄 소개글(`bio`)을 읽어본다.
- 방문자 사용자는 **가장 관심 있는 특정 콘텐츠로 이동하기 위해** 파비콘(아이콘)과 텍스트 제목이 시각적으로 명확해진 개별 링크 버튼 중 하나를 클릭한다.
- 방문자 사용자는 **마이링크 원본 페이지를 잃어버리지 않기 위해** 새 탭으로 무사히 열린 타겟 웹사이트를 확인한 후, 기존 탭으로 돌아와 다른 링크들을 추가로 탐색한다.

## 2. 소유자: 계정 생성 및 프로필 관리 과정
- 소유자 사용자는 **마이링크 서비스를 이용하기 위해** 별도 가입 폼 작성 없이 '구글 소셜 로그인' 버튼을 클릭하여 인증을 진행한다.
- 소유자 사용자는 **자신만의 고유 URL 주소를 최초 배정받기 위해** 시스템이 구글 지메일 아이디의 앞부분을 추출하여 자동 생성한 초기 닉네임(`displayName`)을 확인한다.
- 소유자 사용자는 **원하는 URL 주소로 접속 경로를 변경하기 위해** 대시보드의 닉네임(`displayName`) 텍스트를 클릭하고 자신이 선호하는 고유 단어로 인라인 편집 후 포커스를 해제(칸 바깥 클릭)하여 중복 방지 검사를 마친다.
- 소유자 사용자는 **방문자에게 보여질 실제 이름을 설정하기 위해** 프로필 영역의 유저네임(`username`) 텍스트를 클릭하고 본명이나 활동 채널명으로 자유롭게 인라인 편집한다.
- 소유자 사용자는 **자신을 매력적으로 어필하기 위해** 프로필 소개글(`bio`) 영역을 클릭하고 자신에 대한 설명을 작성한 후 포커스를 해제하여 데이터를 자동 저장시킨다.
- 소유자 사용자는 **자신의 얼굴이나 로고 이미지가 어떻게 적용되었는지 확인하기 위해** 대시보드 상단에 연동된 구글 프로필 이미지 영역을 살핀다.

## 3. 소유자: 링크 목록 관리 과정
- 소유자 사용자는 **새로운 결과물을 사람들에게 공유하기 위해** 대시보드에서 `[새 링크 추가]` 버튼을 클릭하여 리스트 최상단에 비어 있는 새로운 링크 블록을 생성한다.
- 소유자 사용자는 **해당 링크가 어떤 웹사이트인지 방문자에게 알리기 위해** 새로 생성된 링크 블록의 '제목(Title)' 부분을 클릭한 후 텍스트를 타입핑한다.
- 소유자 사용자는 **정확한 타겟 페이지로 방문자를 도달시키기 위해** 'URL' 부분을 클릭한 후 복사해둔 링크 주소를 붙여넣고, 바깥 영역을 클릭해 데이터를 즉각 동기화(저장)한다.
- 소유자 사용자는 **입력한 타겟 URL이 유효하고 시각적으로 돋보이는지 확인하기 위해** 시스템이 구글 퍼블릭 파비콘 API를 통해 자동으로 렌더링한 좌측 아이콘 이미지를 확인한다.
- 소유자 사용자는 **기존에 등록한 링크의 제목 라벨을 더 매력적으로 고치기 위해** 해당 링크의 제목 영역을 클릭해 지체없이 기존 텍스트를 새로운 내용으로 인라인 편집한다.
- 소유자 사용자는 **링크 클릭의 도착지를 다른 페이지 주소로 수정하기 위해** 기존 링크의 URL 텍스트 영역을 클릭해 주소를 변경하고 포커스를 해제하여 수정 내용을 반영한다.
- 소유자 사용자는 **가장 중요한 프로젝트를 방문자가 가장 먼저 볼 수 있게 위로 올리기 위해** 특정 링크 블록 좌측의 드래그 핸들(Drag Handle)을 잡고 위로 혹은 아래로 드래그 앤 드롭하여 노출 순서를 재배열한다.
- 소유자 사용자는 **기간이 종료된 이벤트나 당분간 공유를 끄고 싶은 링크를 방문자 시야에서 감추기 위해** 해당 링크 우측의 `[활성화 토글 스위치]`를 클릭해 Off 상태(비활성화 음영 처리)로 전환한다.
- 소유자 사용자는 **더 이상 내 프로필에 영구적으로 보여주고 싶지 않은 특정 링크 데이터를 제거하기 위해** 해당 링크 블록의 `[휴지통(삭제) 버튼]`을 클릭한다.
- 소유자 사용자는 **실수로 삭제한 링크를 원상 복구하기 위해** 삭제 직후 화면 하단에 일시적으로 나타나는 스낵바 알림 영역의 `[실행 취소(Undo)]` 버튼을 클릭한다.

## 4. 소유자: 방문자 통계 확인 및 페이지 공유 과정
- 소유자 사용자는 **내 프로필이 얼마나 인기 있는지 종합적으로 파악하기 위해** 대시보드의 통계 영역에 표시된 마이링크 페이지의 총 누적 방문 횟수(총 View)를 확인한다.
- 소유자 사용자는 **어떤 콘텐츠가 가장 인기가 좋고 클릭 효율이 높았는지 분석하기 위해** 각 링크 블록 하단이나 상세 통계 창에 표시된 개별 누적 클릭수(Click)를 확인한다.
- 소유자 사용자는 **완성된 내 마이링크 프로필을 인스타그램이나 트위터 등에 공유하기 위해** 대시보드 상단의 고유 서비스 URL(`mylink.com/닉네임`) 옆에 있는 `[URL 복사 버튼]`을 클릭한다.
